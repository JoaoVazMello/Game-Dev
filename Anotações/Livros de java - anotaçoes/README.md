## 📖 Sobre o Livro

Este livro explora os fundamentos do uso da linguagem java de maneira eficiente, falando sobre:  classes, interfaces, exceçoes entre outros assuntos importantes para desenvolvimento que serão abordados aqui.

---

## 🎯 Objetivo do Estudo

Motivo pelo qual escolhi estudar este livro:

- Aprofundar conhecimento na linguagem java

---

## 🧠 Principais Conceitos Estudados

- Arquitetura baseada em componentes
- Separação dos componentes para seguir um design pattern
- Boa praticas para os conceitos de encapsulamento
