# 📚 Anotações

Esta pasta reúne todas as minhas anotações teóricas relacionadas ao desenvolvimento de jogos.

O objetivo não é criar resumos perfeitos, mas sim registrar o conhecimento da forma como eu compreendi os conceitos, reforçando aprendizado e consolidação técnica.

---

## 🎯 Objetivo das Anotações

- Fixar conceitos importantes
- Criar material de revisão futura
- Construir base sólida em fundamentos de Game Engine

---

## 🗂 Estrutura

As anotações são organizadas da seguinte forma:

```
anotacoes/
└── Livro (Assunto do livro)
    ├── (Anotaçao do livro).md
    └── README.md
└── Livro (Assunto do livro)
    ├── (Anotaçao do livro).md
    └── README.md
```
