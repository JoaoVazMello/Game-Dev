# 📘 Nome : Game Engine Architecture, 3rd Edition

Autor: Jason Gregory
Ano: 2018
Área: Arquitetura de Game-Engine

---

## 📖 Sobre o Livro

Este livro explora os fundamentos da arquitetura de game engines modernas, abordando organização de sistemas, gerenciamento de memória, pipelines de renderização e design orientado a dados.

---

## 🎯 Objetivo do Estudo

Motivo pelo qual escolhi estudar este livro:

- Aprofundar conhecimento em arquitetura de jogos
- Entender como engines profissionais são estruturadas
- Aplicar conceitos em projetos próprios

---

## 🧠 Principais Conceitos Estudados

- Arquitetura baseada em componentes
- Separação entre lógica e renderização
- Gerenciamento de memória
- Padrões de projeto aplicados a jogos
- Estruturas de dados voltadas para performance

---

## 🔬 Aplicações Práticas

Como apliquei ou pretendo aplicar o conteúdo:

- Experimentos com organização de memória
- Implementaçao de um prototipo de game engine
- Criaçao de um jogo utilizando alguma game engine
