## 📖 Sobre o Repositório

Este repositório tem como objetivo documentar toda a minha evolução no mundo do desenvolvimento de jogos, reunindo anotações sobre:

- Livros importantes
- Artigos do mercado de games

Além disso, desenvolvo projetos práticos como:

- Mods
- Jogos
- Interfaces
- Mapas
- Game engines

Todos os estudos realizados a partir de livros ou artigos também são documentados de acordo com o conhecimento apresentado.

A estrutura de pasta que contêm essas documentações seguem o mesmo padrão que pode ser visto em [Anotações](Anotações)

Todo o conteúdo é escrito em Markdown e essas anotações são feitas com o objetivo de absorver os conceitos da minha própria maneira, podendo não atender a todos os públicos.

Além disso, o repositório pode incluir outros projetos que contribuam para o meu crescimento profissional.
